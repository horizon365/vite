# 변화를 깨뜨립니다

API 감가 상각, 제거 및 변경 사항을 포함하여 Vite의 중단 변경 목록. 아래 변경의 대부분은 vite 구성에서 [`future` 옵션을](/ko/config/shared-options.html#future) 사용하여 옵트 인 될 수 있습니다.

## 계획된

이러한 변경 사항은 다음 주요 버전의 Vite에 대해 계획됩니다. 감가 상각 또는 사용 경고는 가능한 한 당신을 안내하며, 우리는 프레임 워크, 플러그인 저자 및 사용자에게 이러한 변경 사항을 적용하도록 연락하고 있습니다.

- _아직 계획된 변경 사항이 없습니다_

## 치고는

이러한 변화는 고려되고 있으며 종종 현재 사용 패턴을 개선하려는 실험적인 API입니다. 모든 변경 사항이 여기에 나열되지는 않으므로 [Vite Github 토론의 실험 레이블을](https://github.com/vitejs/vite/discussions/categories/feedback?discussions_q=label%3Aexperimental+category%3AFeedback) 확인하십시오.

아직이 API로 전환하는 것이 좋습니다. 피드백을 수집하는 데 도움이되도록 Vite에 포함되어 있습니다. 이 제안서를 확인하고 각 링크 된 GitHub 토론에서 사용 사례에서 어떻게 작동하는지 알려주십시오.

- [고리에 `this.environment`](/ko/changes/this-environment-in-hooks)
- [HMR `hotUpdate` 플러그인 후크](/ko/changes/hotupdate-hook)
- [환경당 API로 이동하십시오](/ko/changes/per-environment-apis)
- [`ModuleRunner` API를 사용하는 SSR](/ko/changes/ssr-using-modulerunner)
- [빌드 중 공유 플러그인](/ko/changes/shared-plugins-during-build)

## 과거

아래 변경 사항이 수행되거나 되돌아갔습니다. 그들은 더 이상 현재 주요 버전과 관련이 없습니다.

- _아직 과거의 변화가 없습니다_
